## Examples

### API
```rust
use tiktokapi_v2::{
    apis::get_v2_user_info::Api,
    responses::user::UserField,
};
let access_token = "xxx";
let api = Api::new(UserField::all());
let res = api.execute(access_token).await.unwrap();
println!("{:?}", res);
```

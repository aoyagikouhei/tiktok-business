# tiktok-business

TikTok Business API library.

[Documentation](https://docs.rs/tiktok-business)

- OAuth2
- OAuth web example
- Mockable Api

## Supported APIs
- GET /business/comment/reply/create/
- GET /business/get/
- GET /business/video/list/
- POST /business/comment/reply/create/

## Features
### default
- reqwest/default-tls

### rustls-tls
- reqwest/rustls-tls
